Setting up a Scene

First of all you have to understand how to build a context and set up a scene in Crossforge.

We are using ExampleBaseScene as a reference to show you how it can be done:

...\(CrossForge root folder)\Examples\exampleSceneBase.hpp

Crossforge uses different classes where you can set values to create a window or a scene.

  1. We use GLWindow, which is powered by the GLFW library, to set parameters(size and title) for the active window:
//GLWindow m_RenderWin;
m_RenderWin.init(Vector2i(100, 100), Vector2i(m_WinWidth, m_WinHeight), m_WindowTitle);             //init(Position, Size, Title)
m_RenderWin.startListening(this);                                                                   //resizing for changing resolution
  1. We can set up a camera with the class VirtualCamera:
//VirtualCamera m_Cam;
m_Cam.init(Vector3f(0.0f, 3.0f, 8.0f), Vector3f::UnitY());                                          //init(Position, Up-Vector)
m_Cam.projectionMatrix(m_WinWidth, m_WinHeight, GraphicsUtility::degToRad(45.0f), 0.1f, 1000.0f);   
//projectionMatrix(Viewportwidth, Viewportheight, FOV, Near, Far)

As well as light sources with different classes. We use DirectionalLight and PointLight, but you can also use a SpotLight which are all self-explanatory:

//DirectionalLight m_Sun;
//PointLight m_BGLight;
Vector3f SunPos = Vector3f(-5.0f, 15.0f, 35.0f);
Vector3f BGLightPos = Vector3f(0.0f, 5.0f, -30.0f);
m_Sun.init(SunPos, -SunPos.normalized(), Vector3f(1.0f, 1.0f, 1.0f), 5.0f);                         //init(Pos, Direction, Color, Intensity)
// sun will cast shadows
m_Sun.initShadowCasting(1024, 1024, GraphicsUtility::orthographicProjection(10.0f, 10.0f, 0.1f, 1000.0f));
//initShadowCasting(ShadowMapWidth, ShadowMapHeight, Projection)
m_BGLight.init(BGLightPos, -BGLightPos.normalized(), Vector3f(1.0f, 1.0f, 1.0f), 1.5f, Vector3f(0.0f, 0.0f, 0.0f));
  1. The class RenderDevice handles the coordination of render requests and communication with shaders and buffer. We can set render pipeline parameters in RenderDevice::RenderDeviceConfig:
//RenderDevice m_RenderDev;
// configure and initialize rendering pipeline
RenderDevice::RenderDeviceConfig Config;
Config.DirectionalLightsCount = 1;
Config.PointLightsCount = 1;
Config.SpotLightsCount = 0;
Config.ExecuteLightingPass = true;
Config.GBufferWidth = m_WinWidth/m_RenderBufferScale;
Config.GBufferHeight = m_WinHeight/m_RenderBufferScale;	
Config.pAttachedWindow = &m_RenderWin;
Config.PhysicallyBasedShading = true;
Config.UseGBuffer = true;
...
m_RenderDev.init(&Config);

and assign active camera and lights, which we set up previously:

m_RenderDev.activeCamera(&m_Cam);
m_RenderDev.addLight(&m_Sun);
m_RenderDev.addLight(&m_BGLight);
  1. Shaders are managed by the class SShaderManager. We can set parameters to the light shader and post-processing shader with SShaderManager::configShader():
//SShaderManager* m_pShaderMan;
m_pShaderMan = SShaderManager::instance();                                                             //create a new instance of a shader manager
// configure and initialize shader configuration device
//SShaderManager::configShader() can receive either ShaderCode::LightConfig or ShaderCode::PostProcessingConfig as argument
ShaderCode::LightConfig LC;
LC.DirLightCount = 1;                                                                                  //We set 1 directional
LC.PointLightCount = 1;                                                                                //and 1 point light
LC.SpotLightCount = 0;
LC.PCFSize = 1;
LC.ShadowBias = 0.00001f;
LC.ShadowMapCount = 1;
m_pShaderMan->configShader(LC);

//m_RenderDev.init(&Config);

ShaderCode::PostProcessingConfig PPC;
PPC.Exposure = 1.0f;
PPC.Gamma = 2.2f;
PPC.Saturation = 1.1f;
PPC.Brightness = 1.05f;
PPC.Contrast = 1.05f;
m_pShaderMan->configShader(PPC);

(For more functions like updateFPS you can check out exampleSceneBase.hpp)

To run the scene we of course need a main loop:

virtual void run(void) {
			while (!m_RenderWin.shutdown()) {
				m_RenderWin.update();

				m_RenderWin.swapBuffers();

				//updateFPS(); -> check out exampleBaseScene.hpp 

				//defaultKeyboardUpdate(m_RenderWin.keyboard()); -> check out next step in My First Scene(Basic Scene)

			}//while[main loop]
		}//run

And for closing down:

void clear(void) {
			m_RenderWin.stopListening(this);
			if (nullptr != m_pShaderMan) m_pShaderMan->release();
			m_pShaderMan = nullptr;
		}//clear
